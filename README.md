# Smart Voting System DAPP

Прозрачная и безопасная платформа для голосований, построенная на на фреймворке Scaﬀold-ETH 2. Эта система позволяет проводить выборы и опросы с полной прозрачностью.

<img width="2201" height="1166" alt="image" src="https://github.com/user-attachments/assets/f6db98a4-31fa-4f40-9cb2-de006e11a23e" />

## Функционал

- Только владелец может создавать новые голосования
- Простое интуитивное голосование
- Голосования завершаются по истечении времени или принудительно
- Результаты в реальном времени
- Доступ к результатам всех завершенных голосований

## Взаимодействие
- Для начала подключитесь к своему кошельку
- Чтобы создать голосование нужно ввести название, варианты ответа и длительность в соответствующие ячейки
- Далее нужно совершить транзакцию и голосование создано
- <img width="2204" height="1131" alt="image" src="https://github.com/user-attachments/assets/1c79d71d-bfaa-4c06-a8ff-5b3a30203b26" />
- Во вкладке "Проголосовать" находятся все голосования и предварительные результаты
- Чтобы проголосовать, нажмите на нужное голосование и проголосуйте за нужный вариант и совершить транзакцию
- <img width="2255" height="1503" alt="image" src="https://github.com/user-attachments/assets/156e4258-c5d2-4891-b1b4-de3c5dc3b6fc" />
- Также владелец может завершить голосование в любой момент
- По истечении времени голосование завершается
- С результатами всех голосований можно ознакомиться во вкладке "Результаты"
